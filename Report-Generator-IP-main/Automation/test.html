VAPT Report Generator

How to Use This Tool

  • Use the provided Excel template with 15 screenshot columns
  • Add vulnerability details in each row
  • Insert screenshot filenames in the respective "Screenshot X" columns
  • Ensure your steps are separated by line breaks (Enter/Return key)
  • The system will automatically map screenshots to steps in the final report
  • For each vulnerability, the number of steps should match the number of screenshots
  • For an IP having no vulnerabilities, mention it in the Excel file and under "Sr. No." mention as "No Vulnerabilities"
